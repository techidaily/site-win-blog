## Enhancing Usability in Excel: Techniques for Implementing Descriptive Alt Tags on Objects

Alternative text (alt text) allows screen readers to capture the description of an object and read it aloud, providing aid for those with visual impairments. Here’s how to add alt text to an object in Microsoft Excel.

Related: [How to Add Alternative Text to an Object in Google Slides](https://fake-location.techidaily.com/life360-circle-everything-you-need-to-know-on-poco-m6-pro-5g-drfone-by-drfone-virtual-android/) 

 To add alt text to an object in Excel, open your spreadsheet, [add an object](https://fox-boxes.techidaily.com/simplified-steps-for-photo-motion-blur-in-photoshop/) (Insert > Picture), and then select the object.

**![Selected object in Excel](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2020/07/Seleted-object-in-Excel.png)** 

 Right-click the object and then select “Edit Alt Text” from the menu that appears.

**![Edit alt text button on object in Excel](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2020/07/Edit-alt-text-button-on-object-in-Excel.png)** 

 Alternatively, you can select the “Alt Text” option in the “Accessibility” group of the “Picture Format” tab.

**![Alt text tab in Excel](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2020/07/Alt-text-tab-in-Excel.png)** 

 Either option will bring up the “Alt Text” pane in the right-hand side of the window. Unlike the [Alt Text pane in Word](https://eaxpv-info.techidaily.com/updated-2024-approved-from-raw-footage-to-final-cut-youtube-and-its-equivalents/) and [PowerPoint](https://driver-error.techidaily.com/auditory-bliss-achieved-with-win11/), Excel’s lacks the “Generate A Description For Me” option.

 Instead, you’ll need to create the description yourself. The general rule for alt text is to keep it brief and descriptive. You also can omit unnecessary descriptions like “image of” or “photo of” as screen readers announce the object for you.

![Alt text description of object in excel](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2020/07/Alt-text-description-of-object-in-excel.png) 

 If the object is purely decorative, then mark it as so by checking the box next to “Mark As Decorative.” Once selected, the box where you’d normally enter the description is grayed out and you can no longer input a description. The purpose is to let the screen reader know it can safely skip over that object.

**![Mark as descriptive box](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2020/07/Mark-as-descriptive-box.png)** 

 That’s all there is to adding your alt text to objects in Microsoft Excel.
